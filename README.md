# algorithms-in-action

Практические проекты по книге «Грокаем алгоритмы» (2-е изд.). Репозиторий содержит самостоятельные подпроекты, реализующие ключевые алгоритмы из глав.

## Быстрый старт

1. Клонируйте репозиторий, создайте виртуальное окружение и установите зависимости:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
2. Запустите общие тесты для проверки корректности:
   ```bash
   python3 -m pytest tests/
   ```

## Навигация по проектам

Каждая папка содержит изолированный проект со своим описанием и инструкцией по запуску:

* **[sorting_benchmark](./sorting_benchmark)** — Сравнение скорости Bubble, Insertion и Selection Sort на разных типах данных.
* **[maze_pathfinder](./maze_pathfinder)** — Поиск кратчайшего пути в лабиринте с использованием алгоритмов BFS (поиск в ширину) и Дейкстры. *(В разработке)*
* **[knn_recommender](./knn_recommender)** — Простая рекомендательная система на основе алгоритма $k$-ближайших соседей ($k$-NN). *(В планах)*

## Дорожная карта и Прогресс

| Проект | Главы книги | Алгоритмы | Статус |
| :--- | :---: | :--- | :---: |
| **sorting_benchmark** | 1–4 | Сортировки, $O$-большое, QuickSort | Готово |
| **maze_pathfinder** | 6–9 | Графы, BFS, алгоритм Дейкстры | В разработке |
| **knn_recommender** | 12 | Классификация, регрессия, $k$-NN | В планах |

## Стек технологий

* Python 3.11+
* `numpy` (генерация данных и матрицы лабиринтов)
* `pytest` (автоматическое тестирование)

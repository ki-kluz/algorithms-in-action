# Maze Pathfinder

Модуль для генерации идеальных лабиринтов и поиска кратчайшего пути от входа до выхода. Демонстрирует работу DFS (генерация) и BFS (поиск пути) на практике.

## Запуск проекта

Убедитесь, что вы находитесь в корне репозитория и установили зависимости, после чего выполните команду:

```bash
python3 maze_pathfinder/src/main.py
```

## Структура проекта

| Компонент | Назначение |
| :--- | :--- |
| `main.py` | Точка входа: инициализация, генерация, визуализация и запуск поиска пути. |
| `maze.py` | Модуль генерации структуры лабиринта. |
| `pathfinder.py` | Модуль алгоритма поиска кратчайшего маршрута. |
| `structures.py` | Кастомная реализация базовых структур данных. |

## Алгоритмы

### Генерация лабиринта (DFS)

Использует алгоритм поиска в глубину (**Depth-First Search**) с механизмом отката (**Backtracking**).
* **Размеры:** Допускаются только нечётные значения ($$rows \times cols$$, например, $$11 \times 11$$).
* **Точка старта (S):** Верхний левый угол лабиринта.
* **Точка финиша (F):** Нижний правый угол лабиринта.
* **Гарантия:** Между любыми двумя точками существует ровно один путь.

### Поиск пути (BFS)

Использует алгоритм поиска в ширину (**Breadth-First Search**).
* **Входные данные:** Граф смежности лабиринта, координаты старта и финиша.
* **Выходные данные:** Кратчайший путь в виде упорядоченного списка координат.
* **Сложность:** $$O(V + E)$$, где $$V$$ — комнаты (вершины), $$E$$ — проходы (рёбра).

## Пример работы

```
=== Maze generation ===

# # # # # # # # # # #
S   #       #       #
#   #   #   #   #   #
#   #   #       #   #
#   #   # # # # # # #
#   #               #
#   # # # # # # #   #
#           #       #
# # # # #   #   #   #
#               #   F
# # # # # # # # # # #

=== Shortest path search (BFS) ===

# # # # # # # # # # #
S . #       #       #
#   #   #   #   #   #
# . #   #       #   #
#   #   # # # # # # #
# . #               #
#   # # # # # # #   #
# .   .   . # .   . #
# # # # #   #   #   #
#         .   . # . F
# # # # # # # # # # #
```
